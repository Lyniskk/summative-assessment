What is DELTARUNE?

DELTARUNE is a JRPG created by Toby Fox, the same developer behind UNDERTALE. The combat system is almost the same as UNDERTALE, with a [[HeartShapedObject]] acting as the player. Instead of being an actual sequel, it's an alternate universe set in a world that feels familiar, even having characters from UNDERTALE. Unlike UNDERTALE, the game is heavily based on the principle of "Your choices don't matter". For example, killing or sparing the enemies in chapter 1 doesn't change the ending of said chapter.
